# ![TostUp.fun](./assets/logo.png)

**TostUp.fun** is a next-generation AI-powered trading terminal and automated vault management system. It integrates advanced LLM-based market analysis with automated execution strategies, providing users with a "Glass Command Deck" interface for real-time monitoring and control.

## Tech Stack

*   **Frontend:** Next.js 16 (Turbopack), React 19, Tailwind CSS, Framer Motion, Lightweight Charts.
*   **Backend:** Node.js, Drizzle ORM, LangChain, OpenAI.
*   **Infrastructure:** Monorepo structure managed with Turbo.

## Getting Started

### Prerequisites
*   Node.js v20+
*   pnpm or npm

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/sagarjethi/tostup.fun.git
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start development server:
    ```bash
    ./run_dev.sh
    ```
